# 1. DeepSpeech 1.0 (2014)
## 1) 정의
- **RNN 기반의 LSTM/GRU 아키텍처를 사용하는 End-to-End 음성인식 모델**
- **CTC Loss**를 적용하여 입력 음성과 출력 문자 간의 정렬 없이 학습 가능
